## The Logic: Step-by-Step Checklist (UI Flow)

Tahap A: Technical Validation (Bobot 60%)
User harus mencentang/memilih opsi ini di software:

Market Structure (H1):

[ ] Higher High & Higher Low (Uptrend)

[ ] Lower High & Lower Low (Downtrend)

[ ] Sideways / Ranging (Auto-Reject jika dipilih)

Trendline Quality:

[ ] Trendline Major tersentuh minimal 2 kali?

[ ] Apakah harga sekarang sedang retest atau breakout valid?

Trigger (M15):

[ ] Ada candle rejection (pinbar/engulfing) di area Trendline?

[ ] (Opsional) Harga sudah break "Counter Trendline" kecil?

Tahap B: Risk Management (Bobot 20%)
Stop Loss (SL):

[ ] SL jelas di bawah Swing Low/High terakhir?

[ ] Jarak SL masuk akal (tidak terlalu tipis, tidak terlalu lebar)?

Risk Reward Ratio (RR):

[ ] Minimal 1:2 (Software otomatis hitung berdasarkan input harga entry & SL).

Tahap C: Psychology & "The INTP Check" (Bobot 20%)
Ini fitur unik jualanmu. Memaksa user jujur pada diri sendiri.

State of Mind:

[ ] Saya tenang (tidak habis loss beruntun).

[ ] Saya tidak sedang "balas dendam" ke market.

[ ] Saya siap kehilangan uang sebesar Risk (misal $10) di trade ini?

Konfirmasi Akhir:

[ ] "Saya sadar setup ini sesuai SOP, bukan tebak-tebakan."

3. Algoritma "Quality Score" (Fitur Killer)
   Di layar akhir sebelum user klik "Save Plan", software akan memunculkan Score Meter.

Rumus Sederhana:

Setiap centang poin teknis = +10 poin.

RR Ratio < 1:1.5 = -50 poin (Langsung Merah).

State of Mind negatif = -30 poin.

Output Visual:

Score < 50 (Merah): "JANGAN MASUK! Ini Judi, bukan Trading. Tutup Laptop." (Tombol Entry di-disable atau diperingatkan keras).

Score 50 - 70 (Kuning): "Setup Cukup, tapi Berisiko. Kurangi Lot Size 50%."

Score > 80 (Hijau): "Perfect Setup. Execute with Confidence."
